## 介紹
紀錄屬於自己的餐廳清單，可以瀏覽餐廳、查看詳細資訊

## 功能

* 查看所有餐廳
* 瀏覽餐廳詳細資訊
* 搜尋特定餐廳
* 新增餐廳
* 編輯餐廳
* 刪除餐廳

## 開始使用

# 1. 確認有安裝node.js及npm
# 2. 將專案clone至本地
# 3. 在本地開啟後，在終端機輸入


    npm install


## 4.安裝完畢後繼續輸入


    npm run dev


## 5.若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址


    Listening on http://localhost:3000
`


## 6. 若欲停止使用


    ctrl+c
